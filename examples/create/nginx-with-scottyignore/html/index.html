What is .scottyignore?

The .scottyignore file lets you control which files are uploaded when creating a new app with scottyctl app:create.

Files in this example project:

  • Uploaded: docker-compose.yml, html/, .scottyignore
  • Ignored: *.log, .env, node_modules/, *.tmp (and more!)

Try it yourself:

cd examples/create/nginx-with-scottyignore
scottyctl app:create my-demo ./docker-compose.yml --service web:80

Check the .scottyignore file in this directory to see all supported patterns and examples!
